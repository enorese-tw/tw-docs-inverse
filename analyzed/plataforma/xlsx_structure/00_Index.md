# Estructura de Reportes Excel (Extensions)

Este directorio contiene la documentación detallada de la estructura de salida de cada reporte Excel generado en la capa `Teamwork.Extensions`.

## Listado de Reportes

- [XLSX_ReporteAsistencia](XLSX_ReporteAsistencia.md): Reporte de asistencia mensual con cálculos de días trabajados, libres, faltas y licencias.
- [XLSX_ReporteContratos](XLSX_ReporteContratos.md): Reporte de trabajadores con contratos vigentes en un periodo.
- [XLSX_ReporteFiniquitoProcesoMasivo](XLSX_ReporteFiniquitoProcesoMasivo.md): Reporte de estado de procesos masivos de finiquitos.
- [XLSX_ReporteFiniquitos](XLSX_ReporteFiniquitos.md): Reporte detallado y consolidado de finiquitos procesados.
- [XLSX_ReporteFirmaDigital](XLSX_ReporteFirmaDigital.md): Reporte de estado de firmas digitales para modificaciones de cargo.
- [XLSX_ReporteRenovaciones](XLSX_ReporteRenovaciones.md): Reporte de renovaciones de contrato procesadas.
- [XLSX_ReporteSolicitudContratos](XLSX_ReporteSolicitudContratos.md): Reporte detallado de solicitudes de nuevos contratos.
- [XLSX_ReporteSolicitudes](XLSX_ReporteSolicitudes.md): Reporte de solicitudes de baja o modificaciones.
- [XLSX_ReporteTransferencias](XLSX_ReporteTransferencias.md): Archivos para transferencias bancarias masivas (TEF).
- [XLSX_ReporteValeVista](XLSX_ReporteValeVista.md): Reporte para pagos mediante Vale Vista.
