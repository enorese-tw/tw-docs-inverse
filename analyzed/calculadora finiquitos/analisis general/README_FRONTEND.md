# Índice de Análisis Frontend

Documentación generada a partir del análisis de los scripts ubicados en `FiniquitosV2/Scripts/Finiquitos/ajax/`.

## Scripts Principales (Flujo de Negocio)
- [Inicio.js](./Inicio.js.md): Dashboard principal, gestión de estados y anulación de finiquitos.
- [SolicitudBaja.js](./SolicitudBaja.js.md): Formulario de solicitud de baja (inicio del proceso).
- [VisualizacionCalculoFiniquito.js](./VisualizacionCalculoFiniquito.js.md): Detalle del cálculo, promedios y generación de PDF.
- [XmlSolB4J.js](./XmlSolB4J.js.md): Bandeja de entrada y flujos de aprobación de solicitudes.

## Gestión de Pagos
- [BasePagosFiniquitos.js](./BasePagosFiniquitos.js.md): Listado y filtros de pagos a trabajadores.
- [BasePagosProveedores.js](./BasePagosProveedores.js.md): Listado y filtros de pagos a proveedores.
- [PagoProveedores.js](./PagoProveedores.js.md): Generación masiva de cheques para proveedores.
- [PagosFiniquitos.js](./PagosFiniquitos.js.md): Validación de documentos de pago.
- [AdministradorDeProveedores.js](./AdministradorDeProveedores.js.md): ABM de proveedores.

## Utilidades y UI
- [Utils.js](./Utils.js.md): Funciones de formateo (RUT, Fechas, Moneda).
- [Loader.js](./Loader.js.md): Plugin jQuery para spinner de carga.
