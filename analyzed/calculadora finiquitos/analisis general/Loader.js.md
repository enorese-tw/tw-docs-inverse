# Análisis de Loader.js

## Ubicación
`FiniquitosV2/Scripts/Finiquitos/ajax/Loader.js`

## Descripción
Plugin de jQuery simple (`$.fn.loader`) para mostrar y ocultar una cortina de carga (spinner) en la interfaz de usuario.

## Funciones Principales
- `init`: Inyecta el HTML del loader y lo muestra.
- `stop`: Oculta el loader y limpia el HTML.

## Llamadas al Servidor
- **Ninguna**. Es un componente puramente visual.

## Lógica de Negocio
- No contiene lógica de negocio del dominio. Solo lógica de presentación UI.
